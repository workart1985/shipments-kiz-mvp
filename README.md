## Shipments + KIZ (MVP)
Next.js + Supabase RPC. Скан "как клавиатура" + ввод КИЗ.

### ENV
cp .env.local.example .env.local
# заполни значения Supabase

### SQL схема
Открой Supabase → SQL Editor → вставь содержимое файла `supabase.sql` и выполни.

### Run локально
npm i
npm run dev
# http://localhost:3000
